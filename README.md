# 🍴 SISTEMA RESTAURANTE

Sistema de gestión para restaurantes desarrollado en **PHP y MySQL**.  
Permite administrar **pedidos, platos, tiempos de preparación y balances gráficos**.  
Los clientes pueden **realizar pedidos y consultar sus boletas**, mientras que el personal tiene **control total del inventario y las operaciones del restaurante**.  
Ideal para negocios que buscan organizar sus procesos de manera eficiente y visual.

---

## 🛠 Funcionalidades principales
- ✅ Registro y gestión de **mesas**  
- ✅ Registro de **pedidos** y **platos**  
- ✅ Control de **tiempo estimado** de preparación  
- ✅ Gestión de **stock e inventario**  
- ✅ Generación de **balances y reportes gráficos**  
- ✅ Interacción con clientes: realizar pedidos y consultar **boletas**  

---

## 💻 Tecnologías utilizadas
- **PHP**  
- **MySQL**  
- **HTML / CSS / JavaScript**
